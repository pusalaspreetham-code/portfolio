# 🚀 Preetham Portfolio

A modern personal portfolio website built using React and Tailwind CSS to showcase my projects, hackathons, and internships.

---

## 🌐 Live Demo
👉 https://preethampusala-portfolio.vercel.app/

---

## 📌 Features

* Projects section with live demo & GitHub links
* Hackathons and internships showcase
* Responsive design (mobile + desktop)
* Clean and minimal UI
* Dark mode support

---

## 🛠️ Tech Stack

* React
* Tailwind CSS
* JavaScript

---

## 📂 Folder Structure

* `components/` → reusable UI components
* `sections/` → main sections (About, Projects, etc.)
* `data/` → project, hackathon, internship data

---

## ▶️ Run Locally

```bash
npm install
npm start
```

---

## ☁️ Deployment

Deployed using Azure Static Web Apps.
